
## The new file extension is ".huh"!

---

Try the following ways to use the new file extension:

```bash
git clone https://github.com/yigitkabak/huh
```

Set up packages now:

```bash
cargo build
```

Set up the `file.txt` file and start the project:

```bash
cargo run
```
